Trabalhar com o Amazon EBS
Visão geral do laboratório
O Amazon Elastic Block Store (Amazon EBS) é um serviço de armazenamento em bloco dimensionável e de alto desempenho projetado para o Amazon Elastic Compute Cloud (Amazon EC2). Neste laboratório, você aprenderá a criar um volume do EBS e a executar operações nele, como anexá-lo a uma instância, criar um sistema de arquivos e fazer um backup de snapshot.

 

Diagrama esquemático mostrando uma instância do EC2 com um volume do EBS anexado e um snapshot criado com base no volume do EBS.

 

Objetivos
Ao final deste laboratório, você poderá:

Criar um volume do EBS.

Anexar e montar o volume do EBS em uma instância do EC2.

Criar um snapshot de um volume do EBS.

Criar um volume do EBS com base em um snapshot.
